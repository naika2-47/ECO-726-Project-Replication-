# Replication Project — Measles Vaccination and Economic Outcomes

**Author:** Naika Jean Baptiste  
**Course:** ECO 726  

This repository contains replication materials for *“The Long-Term Effects of Measles Vaccination on Earnings and Employment”* by Atwood (2022).  
The project replicates Figures 1 and 2 from the paper using Stata.

## Contents
- `replication_figures.do`: Stata code for generating Figures 1 and 2  
- `replication_figure1.log`, `replication_figure2.log`: Log files from Stata runs  
- `case_counts_population_raw.csv`, `incidence_rates.csv`: Exported datasets
